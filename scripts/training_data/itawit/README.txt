# Itawit Training Data

Place your Itawit audio reference files here for voice cloning.

## Supported Formats
- WAV (recommended)
- MP3
- FLAC

## Instructions
1. Place audio files (10-60 seconds) of Itawit speech in this directory
2. The TTS system will automatically use these as reference for Itawit language generation
3. Recommended: Clear audio with one speaker, minimal background noise

## Sample Itawit Phrases for Training
- "Mabalin" (Hello)
- "Agyamanak" (Thank you)  
- "Kayat nga" (Yes)
- "Saanno" (When)
- "Indi" (No)
- "Sadya" (Good)
- "Napintas" (Beautiful)
